# Project name
STM32를 사용한 DotMatrix제어

# Project Purpose
1. STM32 구조와 Datasheet 해석 능력 향샹
2. STM32CubeIDE 툴 사용능력 향상
3. GPIO 및 Dot Matrix 제어방법 학습

# Requirement

## Tool
STM32CubeIDE 1.12.1

## Kit 
HBE-MCU-Multi II M4

## Chipset 
Main Chipset : CortaxM4 - STM32F405RGTx
sub Chipset : LD1071
